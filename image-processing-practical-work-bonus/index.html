Thresholding¶

Using the histogram, one can see that there are 3 peaks. The left peak is the darkest one and corresponds to the colors of the dice bodies. We can cut around 120 to isolate the dice bodies from the pips on the dice.

Further reading: Thresholding (Scikit)

import ipywidgets as widgets
import matplotlib.pyplot as plt
import numpy as np

# set up plot
fig, ax = plt.subplots(figsize=(6, 4))


@widgets.interact(low_tresh=(0, 255, 1))
def update(low_tresh=120):
    """Remove old lines from plot and plot new one"""
    plt.cla
    Gb = np.zeros_like(G)
    Gb = np.where(G > low_tresh, 1, Gb)
    ax.imshow(Gb, cmap="binary")
